World Plastic Pollution Dataset

We used this dataset to create our main map that shows the total plastic waste per capita in 2016. First, we cleaned the data by getting rid of any null values and then converted the csv file to a json/js file for easier compatability. We organized the data further by getting rid of feature names we didn't need, like rank. This dataset includes multiple identifying features including cca3 country codes, country names, as well as regions. However, it was missing the longitude and latitude values, which we added in through a Python function.
